# 🚀 Overview

Nova AI is a modern desktop AI voice assistant developed using Python.

It enables users to control their computer using natural voice commands, automate everyday tasks, launch applications, browse the web, play music, and interact with an intelligent AI assistant through a clean desktop interface.

Nova AI is currently under active development and new features are continuously being added.

---

# 📦 Windows Installation

## Download

Download the latest Windows version from GitHub Releases:

👉 https://github.com/sauravkr-dev/Nova-AI-Voice-assistant/releases/latest

---

## Installation Steps

1. Download **NovaAI.exe**
2. Double-click the executable.
3. If Windows SmartScreen appears:
   - Click **More info**
   - Click **Run anyway**
4. Grant microphone permission if requested.
5. Launch Nova AI.

---

## System Requirements

- Windows 10 / Windows 11
- 64-bit Operating System
- 4 GB RAM or higher
- Microphone
- Internet connection (for online features)

# 💻 Build From Source

Clone the repository

```bash
git clone https://github.com/sauravkr-dev/Nova-AI-Voice-assistant.git
```

Go to the project folder

```bash
cd Nova-AI-Voice-assistant
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Nova

```bash
python main.py
```
